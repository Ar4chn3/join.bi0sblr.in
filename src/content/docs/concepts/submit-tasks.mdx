### How to submit tasks
Format your "tasks" directory in such a manner that the files and folders are organised as shown below. Add a screenshot of the output of the tree command as well as a README.md in your directory along with Tasks
The README.md file should contain: 
> Your name

> You College email 

> Your Discord ID
